# Калькулятор на Ruby

Простой калькулятор с графическим интерфейсом, написанный на Ruby с использованием библиотеки Tk.

## Установка

1. Убедитесь, что у вас установлен Ruby
2. Установите зависимости:
```bash
bundle install
```

## Запуск

```bash
ruby calculator.rb
```

## Функциональность

- Сложение
- Вычитание
- Умножение
- Деление
- Работа с десятичными числами
